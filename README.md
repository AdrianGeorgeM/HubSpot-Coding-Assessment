# HubSpot Coding Assessment

This project is part of the HubSpot Coding Assessment. It's designed to solve a specific problem statement related to organizing a special two-day event for HubSpotâ€™s partners across different countries. The solution involves processing data received from an API, finding suitable dates for the event in each country, and then submitting this information back to another API.

## Getting Started

These instructions will help you set up the project on your local machine for development and testing purposes.

### Prerequisites

What you need to install the software:

- [Node.js](https://nodejs.org/)
- npm (comes installed with Node.js)

### Installing

1. Clone the repository

```
git clone
```

2. Install dependencies

```
npm install
```

3. Run the application

```
npm start
```
